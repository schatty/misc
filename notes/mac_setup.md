1. Download iTerm2
  * download Github.itermcolors colours config

  * in the `General` tab apply current settings to `Default` profile
2. Download vim, .vimrc
3. Install zsh
4. Set UTF-8
  `export LC_CTYPE=en_US.UTF-8`
  `export LC_ALL=en_US.UTF-8`
